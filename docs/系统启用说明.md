# 系统启用说明

## 环境准备

```bash
python -m pip install uv  # 已安装时跳过
uv sync --all-groups
cp .env.example .env
```

本地默认使用 SQLite。使用 MySQL 8 时在 `.env` 中设置 `DB_ENGINE=mysql` 及对应的 `DB_*` 变量。

## 本地启动

```bash
uv run python manage.py migrate
uv run python manage.py createsuperuser
uv run python manage.py runserver
```

- 主页：<http://127.0.0.1:8000/>
- 后台：<http://127.0.0.1:8000/admin/>

启动前检查项目：`make check`。
